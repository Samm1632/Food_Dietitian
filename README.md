# Calorie Intake and Recipe Recommendation

This project is a web application built with Streamlit that helps users calculate their daily recommended calorie intake and provides recipe suggestions based on uploaded dietary data. It also uses OpenAI's language model to provide detailed recipe insights.

---

## Features

1. **Calorie Calculation**:
   - Calculates daily calorie intake based on the Mifflin-St Jeor Equation.
   - User inputs: height, weight, age, and gender.

2. **Recipe Recommendations**:
   - Dynamically loads recipes from a CSV file.
   - Displays macronutrient breakdown (protein, carbs, fat) for each recipe.

3. **LLM-Powered Recipe Insights**:
   - Generates detailed recipe information and tips using OpenAI's GPT model.
